# EasyVista IT Department Integrations

Integration toolkit for EasyVista ITSM platform.

## Platform Components

| Component | Purpose |
|-----------|---------|
| EV Service Manager | ITSM solution (Incident, Problem, Change, Service Request) |
| EV Service Apps | Custom portals and dashboards |
| EV Self Help | Knowledge base and virtual agents |
| EV Observe | Infrastructure and application monitoring |
| EV Digital Experience Monitoring | Real user experience tracking |
| EV Discovery | Automated asset discovery and CMDB |
| EV Reach | Remote IT support |
| EV Orchestrate | IT automation and workflow orchestration |
| EV Pulse AI | AI-powered assistance across all modules |
